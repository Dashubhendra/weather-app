

 Weather App

 Project Overview

The Weather App is a simple and responsive web application that provides real-time weather information for any city entered by the user. It fetches live weather data using a public weather API and displays details such as temperature, humidity, wind speed, and weather conditions in a user-friendly interface.

 Technologies Used

HTM – Structure of the web application
CSS– Styling and responsive design
JavaScript – Fetching API data and dynamic content updates

 Features

* Search weather by city name
* Displays current temperature, humidity, and wind speed
* Shows weather condition with icons
* Responsive design for different screen sizes
* Real-time data fetched from a weather API

Learning Outcomes
* Working with APIs using JavaScript
* Handling asynchronous operations with `fetch()`
* DOM manipulation and event handling
* Building responsive layouts using CSS
